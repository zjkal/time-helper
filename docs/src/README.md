## 安装

::: info
版本要求: PHP >= 7.1
:::

安装非常简单，因为它是一个标准的 [Composer](https://getcomposer.org/) 包，直接使用以下命令安装。

```bash
composer require zjkal/time-helper
```

## 更新
```bash
composer update
```
